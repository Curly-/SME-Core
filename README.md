# Building and installing
## Install dependencies

### Ubuntu
```
sudo apt-get install -y libglm-dev
```

### Arch
```
sudo pacman -S glm
```

## Install SME\_Core
```
git clone https://github.com/Phase-Matrix-Software/SME-Core
cd SME-Core
make
make install
```
